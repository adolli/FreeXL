example: 003

Purpose:

Create region from a bitmap

 
OS:

windows (a SDK project)

 
Place:

example/003/src

 
Notice:

 

 
Usage:

use FCWin32::ConvertAlphaImageToHRGN to create HRGN from a 32bpp image.

This function determinate whether a point in region by test pixel's alpha value, if the pixel's alpha value == 0, the point outter of region, otherwise point in region. and the image's R-G-B channel be ignored

this demo load a tga image (in resource) then convert to 8bpp, as a 32bpp image's alpha channel
